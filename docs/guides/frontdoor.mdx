zrok frontdoor is a way of using zrok-as-a-service from [zrok.io](https://zrok.io) as a shielded entry point to your website or service. This is useful if you want to expose a service to the public internet, but don't want to expose the service directly.

  <TabItem value="Linux">

On Linux, zrok frontdoor is implemented natively as a system service provided by the `zrok-share` DEB or RPM package.

If you'd prefer to run zrok in Docker, you can follow the same Docker instructions for [macOS](./?os=Mac+OS) or [Windows](./?os=Windows).

  </TabItem>
